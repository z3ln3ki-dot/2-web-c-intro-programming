Proyecto Final de Desarrollo Web 1

Este repositorio contiene el proyecto final para el curso de introducción al desarrollo web. El objetivo principal ha sido construir una página web utilizando exclusivamente HTML semántico, sin el uso de hojas de estilo (CSS), para demostrar la importancia de una correcta estructura de documentos en la web.

Descripción del Proyecto:

La página web es un portafolio personal básico que presenta información sobre el autor, sus intereses y medios de contacto. Se ha puesto énfasis en el uso de etiquetas que aportan significado al contenido (semántica) en lugar de solo apariencia.

Requisitos Técnicos Cumplidos(la matoria)

El proyecto integra los siguientes elementos requeridos:

Estructura Semántica: Se han utilizado las etiquetas header, nav, main, section y footer para organizar lógicamente el contenido.

Jerarquía de Texto: Uso correcto de encabezados (h1, h2) y párrafos (p) para facilitar la lectura y accesibilidad.

Multimedia: Inclusión de imágenes utilizando la etiqueta img con sus respectivos atributos de texto alternativo (alt) y dimensiones (width, height).

Listas: Implementación de listas desordenadas (ul) para los hobbies y listas ordenadas (ol) para las metas.

Navegación: Uso de enlaces internos (anclas) para navegar entre las diferentes secciones de la página.

Separación Visual: Uso de la etiqueta hr para delimitar temáticamente las secciones ante la ausencia de estilos visuales.
